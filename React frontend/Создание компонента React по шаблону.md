# Создание компонента React по шаблону

```shell
yarn add generate-react-cli -D
```

добавить в package.json
```json
"scripts": {
    "comp": "generate-react component"
  },
```

теперь создаем компонент 
```bash
yarn comp
```

в корне проекта создаем папку templates
